# .ai-memory — repo adapter for the Nexus memory system

This folder is a thin adapter, **not** the memory store. The store is an external Obsidian vault (`50-Memory/`) synced via Obsidian Sync.

- `repo-profile.md` — committed, repo-safe profile (mirrors the vault copy; regenerated by `memory-export`).
- `memory-map.md` — committed pointers into the vault.
- `local/` — **gitignored**; holds the actual vault path and any private imports.

Agents read the committed files; propose durable memories as `memory-candidate` blocks (see `AGENTS.md`). Never commit the vault or secrets.
